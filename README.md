# Next Shop
This is just a test of a static product store built to learn NextJS. It is powered by Platzi and the Shopify API.

<img src="/docs/home.png" width="750px" alt="home-screenshot">


## ✅ Features

- Show Main products
- Show products
- Show categories
- Show collections
- Get product from Shopify API

## ⚙️ Tech Stack
- NextJS 14.2.3
- SCSS
- Shopify API
- graphql-request 7

## 💾 Installation

Install and run

1. Clone and move to folder
```bash
$ git clone git@github.com:abrahamuchos/next-shop.git
$ cd next-shop
```

2. Install dependencies
```bash
$  npm install
```

3. Run dev `npm run dev` or run to production `npm run build`

*Note: You need account in Shopify and create Store.*
